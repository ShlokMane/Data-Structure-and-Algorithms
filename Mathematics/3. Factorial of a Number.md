# Factorial of a Number

Assumption is n would always be greater than equal to 0.

Example: 

1. I/P: 4
   O/P: 24
2. I/P: 6
   O/P: 720

n! = 1*2*3 ... (n-1)*n

### Iterative Solution:

```java

int fact(int n) {
    int res = 1;
    for(int i = 2;i<=n;i++) {
        res = res*i;
    }
    return res;
}
```

Time Complexity: θ(n)
Auxiliary Space: O(1)

### Recursive Solution: 

```java

int recursiveFact(int n) {
    if(n<=1) {
        return 1;
    }
    return n * recursiveFact(n-1);
}
```

Time Complexity: θ(n)
Auxiliary Space: O(n) // Recursive stack will take space
