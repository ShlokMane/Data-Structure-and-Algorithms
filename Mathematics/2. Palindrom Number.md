# Palindrome Number

Given a number n check if it's a palindrome or not. n will always be greater than equal to 0

Example

1. I/P: 78987
   O/P: Yes
2. I/P: 9
   O/P: Yes
3. I/P: 123
   O/P: No

Idea: Traverse digits from right to left and find out the reverse of the given number if they are same then given number
is a palindrome.
Last digit = n%10, and remaining = n/10;

```java

boolean isPalindrome(int n) {
    int rev = 0, temp = n;

    while (temp > 0) {
        rev = rev * 10 + temp % 10;
        temp = temp / 10;
    }

    return (rev == n);
}
```

Time Complexity: θ(d) , where d is number of digits in n.
