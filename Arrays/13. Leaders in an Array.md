Given an array print the elements who does not have anything greater than them on their right side of the array.

```
Example - 
I/P - arr = [7, 10, 4, 3, 6, 5, 2]
O/P - 10 6 5 2
```

### Naive Solution

Traverse the array and for every element check if anything is greater than it on the right side is present.

```java
void leaders(int[] arr, int n) {
    for (int i = 0; i < n; i++) {
        boolean flag = true;
        for (int j = i + 1; j < n; j++) {
            if (arr[i] <= arr[j]) {
                flag = false;
                break;
            }
        }
        if (flag) {
            System.out.print(arr[i] + " ");
        }
    }
}
// Time Complexity - O(n²)
```

### Efficient Solution

We traverse the array from right to left and maintain a max element variable, if current element is greater than 
to it then it is a leader.

```java
void leaders(int[] arr, int n) {
    int max = arr[n - 1];
    System.out.print(arr[n - 1] + " ");
    for (int i = n - 2; i >= 0; i--) {
        if (arr[i] > max) {
            System.out.print(arr[i] + " ");
            max = arr[i];
        }
    }
}
// Time Complexity - θ(n)
```
