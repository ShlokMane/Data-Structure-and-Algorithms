We are given an array with stock prices for next n days. We have to make the most profit we can make with it only one
stock bought at any given point of time.

```
Example -
I/P - arr[] = [1, 5, 3, 8, 12]
O/P = 13 (Buy at 1 sell at 5, buy at 3 sell at 12)

I/P - arr[1, 5, 3, 1, 2, 8]
O/P - 11 (Buy at 1 sell at 5, buy at 1 sell at 8)
```

### Naive Solution

A recursive solution where we take the array, start and end. For every start and end we try to find a pair such that
`arr[start]<arr[end]` and then call the same function for left and right of those pair. We do it for all such pairs
and return the maximum price this start and end combination can provide.

```java
int maxProfit(int[] arr, int start, int end) {
    if (start >= end) {
        return 0;
    }
    int profit = 0;
    for (int i = start; i < end; i++) {
        for (int j = i + 1; j <= end; j++) {
            if (arr[j] > arr[i]) {
                int curr_profit = arr[j] - arr[i] + maxProfit(arr, start, i - 1) + maxProfit(arr, j + 1, end);
                profit = Math.max(curr_profit, profit);
            }
        }
    }
    return profit;
}
// Time Complexity - O(n²)
```

### Efficient Solution

If we think closely when will we buy the stock? Answer is when the price is at the lowest, and we will sell them when
they are at their peak. After the peak if it starts going down again we won't be buying the stock until it hits the
lowest and follow the same pattern.

So we can say if previous day price is less than current day we will buy the stock and sell it today.

```java
int maxProfit(int[] arr, int n) {
    int profit = 0;
    for(int i =1;i<n;i++) {
        if(arr[i-1]<arr[i]) {
            profit = arr[i]-arr[i-1];
        }
    }
    return profit;
}
// Time Complexity - O(n)
```
