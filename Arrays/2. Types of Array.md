There are two types of array -

1. Fixed Size Array
2. Dynamic Size Array

How to define fixed size array in java -

```java
int[] arr = new int[100];
int[] arr = new int[n];
int arr[] = {10, 20, 30, 40};
```

Dynamic Sized Arrays are defined using `ArrayList` in java.

```java
import java.util.*;

List<Integer> arrayList = new ArrayList<>();
```

This arrays also have a fixed size and once that size is full depending on language the size is doubled or tripled. This
increase in size would require copying of all elements to a new address where that much contiguous free space is
available. This is just and simple implementation there might be other implementation.

### Insert an element in Dynamic Size Array

Time complexity for inserting first n element is `O(1)`

Time complexity for inserting n+1 element is θ(n) as we need to copy all element from one place to another and then add
the element at the end.

Average Time complexity for n+1 inserts = $\frac{O(1)+O(1)+...+O(1)+θ(n)}{n+1} = O(1)$
