There are two types of array -

1. Fixed Size Array
2. Dynamic Size Array

How to define fixed size array in java -

```java
int[] arr = new int[100];
int[] arr = new int[n];
int arr[] = {10, 20, 30, 40};
```

Dynamic Sized Arrays are defined using `ArrayList` in java.

```java
import java.util.*;

List<Integer> arrayList = new ArrayList<>();
```

This arrays also have a fixed size and once that size is full depending on language the size is doubled or tripled. This
increase in size might require copying of all elements to a new address where that much contiguous free space is
available. This is just and simple implementation there might be other implementation.
