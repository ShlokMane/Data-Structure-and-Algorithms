Give an array and a number d, we have to rotate the array counter-clockwise that many times. We can assume d<arr.length.

```
Example - 
I/P - arr[] = [1, 2, 3, 4, 5]
      d = 2
O/P - arr[] = [3, 4, 5, 1, 2]
```

NOTE - Even if d>= arr.length we can do d=d%n.

### Naive Solution - 1

We call `leftRotateOne` function d times.

```java
void leftRotateOne(int[] arr, int n) {
    int temp = arr[0];
    for (int i = 1; i < n; i++) {
        arr[i - 1] = arr[i];
    }
    arr[n - 1] = temp;
}

void leftRotate(int[] arr, int n, int d) {
    for (int i = 0; i < n; i++) {
        leftRotateOne(arr, n);
    }
}
// Time Complexity - θ(n*d)
// Space Complexity - θ(1)
```

### Naive Solution - 2

We copy the first d elements to an temp array. Then we shift all the elements from d to n-1 to 0,1... and then finally
copy the elements from temp array at the end of the given array.

```java
void leftRotate(int[] arr, int n, int d) {
    int[] temp = new int[d];
    for (int i = 0; i < d; i++) {
        temp[i] = arr[i];
    }
    for (int i = d; i < n; i++) {
        arr[i - d] = arr[i];
    }
    for (int i = 0; i < d; i++) {
        arr[n - d + i] = temp[i];
    }
}
// Time Complexity - θ(n)
// Space Complexity - θ(d)
```

### Efficient Solution

If you look closely in Naive Solution 2 we are pushing the elements after d to front and first d elements then to back
of the array. So to achieve this without an temp array we would first reverse elements from 0 to d-1. Now if you think
for a second that if we reverse the whole array now these first d elements would be placed at the end as they should be
after the rotation.

But the elements after d are placed at the wrong position.
```
Example -
I/P - arr[] = [1, 2, 3, 4, 5]
      d = 2
Expected- arr[] = [3, 4, 5, 1, 2] 
We reverse first d elements. 
arr[] = [2, 1, 3, 4, 5]
Now if we reverse whole array - [5, 4, 3, 1, 2]
```
If you look closely the elements at the start are just reversed than their expected position.

So before reversing the whole array let's reverse the elements from d to n-1;

```
Example -
Step 1 - We reverse first d elements. 
         arr[] = [2, 1, 3, 4, 5]
Step 2 - We reverse elements from d to n-1
         arr[] = [2, 1, 5, 4, 3]
Step 3 - Now if we reverse whole array - [3, 4, 5, 1 ,2]
```

```java
void leftRotate(int[] arr, int n, int d) {
    reverse(arr, 0, d-1);
    reverse(arr, d, n-1);
    reverse(arr, 0, n-1);
}

void reverse(int[] arr, int low, int high) {
    while(low<high) {
        int temp = arr[low];
        arr[low] = arr[high];
        arr[high] = temp;
        low++;
        high--;
    }
}
// Time Complexity - θ(n)
// Space Complexity - θ(1)
```
