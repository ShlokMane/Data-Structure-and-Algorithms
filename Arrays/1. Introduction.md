An array is a linear data structure storing a collection of fixed-size, similar-typed elements in contiguous memory
locations.

Example:

```
arr[] = [1, 2, 3, 4, 5];
```

How is array different from other data structures like linked list, hash etc. is elements are store at contiguous
location. So if first element is at location `x` and size of every element is 'y' then our second element would be at
`x+y`, third element at `x+2y` and so on.

Since elements are at contiguous location we can create a formula to know the location of n'th element. That formula
is `address of n'th element = x+(n-1)y`. Since we have a formula to get the address of an element in array we use
indexes in array to get elements. In most of the languages indexes begins with 0 so formula for accessing index would
become `address of i'th element  = x + i*y`.

Advantages of this contiguous location -

1. Random Access of elements because we have a direct formula to point to the address of any element.
2. Cache Friendliness - since elements are at contiguous location when an element at location i'th is fetched the
   processor will fetch nearby element as well assuming that those might be required too.

Array is inside heap i.e. RAM of the system. And the Cache we are talking about above is CPU Cache which is present
inside the processor chip directly.
