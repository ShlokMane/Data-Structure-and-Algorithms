Remove duplicates from a sorted array, and after removal the array should still be sorted.

```
Example - 
arr[] = [10, 20, 20, 30, 30, 30, 30]
n = 7
O/P = [10, 20, 30, _, _, _, _]
n = 3
```

### Naive Solution

Create a temp array and only copy distinct elements into that array.

```java
int removeDuplicate(int[] arr, int n) {
    int[] temp = new int[n];
    temp[0] = arr[0];
    int temp_size = 1;
    for (int i = 1; i < n; i++) {
        if (arr[i] != temp[temp_size - 1]) {
            temp[temp_size] = arr[i];
            temp_size++;
        }
    }
    for (int i = 0; i < temp_size; i++) {
        arr[i] = temp[i];
    }
    return temp_size;
}
// Time Complexity - O(n)
// Space Complexity - O(n)
```

### Efficient Solution

Instead of creating a temp array we would track temp_size in the current array only.

```java
int removeDuplicate(int[] arr, int n) {
    int temp_size = 1;
    for(int i =1;i<n;i++) {
        if(arr[i] != arr[temp_size-1]) {
            arr[temp_size] = arr[i];
            temp_size++;
        }
    }
    return temp_size;
}
// Time Complexity - O(n)
// Space Complexity - O(1)
```
