Given an array check if it's sorted or not.

### Naive Method

For every element check if any of the previous element is greater than it or not. If yes then it's not sorted.

```java
boolean isSorted(int[] arr, int n) {
    for(int i =1;i<n;i++) {
        for(int j=0;j<i;j++) {
            if(arr[j]>arr[i])
                return false;
        }
    }
    return true;
}
// Time Complexity - O(n²)
```

### Efficient Approach

For every element just check the previous element it should not be greater than current element.

```java
boolean isSorted(int[] arr, int n) {
    for(int i =1;i<n;i++) {
        if(arr[i-1]>arr[i]) 
            return false;
    }
    return true;
}
// Time Complexity - O(n)
```
