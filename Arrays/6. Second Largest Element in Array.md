Given an array find the second-largest element in array.

```
Example - 
arr[] = [10, 5, 8, 20]
O/P = 0

arr[] = [10,10,10]
O/P = -1 
No second largest element
```

### Naive Approach

We first find the largest element in the array and then again we find the largest element which is not equal to the
largest we found.

```java
int secondLargest(int[] arr, int n) {
    int largest = getLargest(arr, n);
    int res = -1;
    for (int i = 0; i < n; i++) {
        if (arr[i] != arr[largest]) {
            if (res == -1)
                res = i;
            else if (arr[i] > arr[res])
                res = i;
        }
    }
    return res;

}
// Time Complexity - O(n)
```

* Function getLargest is from [Largest Element in an Array](Largest Element in an Array.md) Notes

This is a naive approach because it requires two traversal of the array.

### Efficient Approach

We keep two variable res and largest to store index of the second largest and largest respectively we update them as we
traverse the array.

```java
int secondLargest(int[] arr, int n) {
    int res = -1, largest = 0;
    for (int i = 1; i < n; i++) {
        if (arr[i] > arr[largest]) {
            res = largest;                         // Current largest will become second largest
            largest = i;
        } else if (arr[i] != arr[largest]) {
            if (res == -1 || arr[i] > arr[res])
                res = i;
        }
    }
    return res;
}
```
