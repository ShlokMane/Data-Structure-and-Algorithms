## Searching an element in an array

Given an array find the index of first occurrence of an element. If element is not present return -1


```java
int search(int[] arr, int n, int x) {
    for(int i = 0; i<n;i++) {
        if(arr[i] == x) {
            return i;
        }
    }
    return -1;
}

// Time Complexity - O(n).
```

## Insert an element into the array

Given an array add the element at xth position. NOTE : The array should have an empty space. 

```java
/***
 * Example - arr[5, 10, 20, _, _]
 * x = 7, position = 2
 * O/P - arr[5, 7, 10, 20, _]
 */

/***
 * cap -> capacity of the array,  
 * n -> number of element present in the array, 
 * pos -> position at which element is to be added, 
 * x -> element
 * returns : number of element present in the array
 */

int insert(int[] arr, int n, int x, int cap, int pos) {
    if (n==cap) {
        return n;
    }
    int idx = pos - 1;
    
    for(int i =n-1;i>=idx;i--) {
        arr[i+1] = arr[i];
    }
    arr[idx] = x;
    return n+1;
}
// Time Complexity - O(n).
```

### Deletion of an element from the array

Given an array remove the first occurrence of an element.

```java
/***
 * Example - arr[3, 8, 12, 5, 6]
 * x = 12
 * O/P - [3, 8, 5, 6, _]
 */

int deleteElement(int[] arr, int n, int x) {
    int i;
    for (i = 0; i < n; i++) {
        if (arr[i] == x)
            break;
    }
    if (i == n) {
        return n;
    }

    for (; i < n - 1; i++) {
        arr[i] = arr[i + 1];
    }
    return n - 1;
}

// Time Complexity - θ(n)
```
