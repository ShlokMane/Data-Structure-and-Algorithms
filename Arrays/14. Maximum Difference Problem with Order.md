Maximum value of `arr[j] - arr[i]` such that `j>i`

```
I/P - arr[] = [2, 3, 10, 6, 4, 8, 1]
O/P - 8
```

### Naive Solution

Keep maximum difference variable. Traverse the whole array for each element check all the element on the right and check
maximum difference.

```java
int maxDiff(int[] arr, int n) {
    if (n == 1) {
        return 0;
    }

    int res = arr[1] - arr[0];
    for (int i = 0; i < n - 1; i++) {
        for (int j = i + 1; j < n; j++) {
            res = Math.max(res, arr[j] - arr[i]);
        }
    }
    return res;
}
// Time Complexity - O(n²)
```

### Efficient Solution

For any element the maximum difference it can bring is by subtracting it by min value on its left.

```java
int maxDiff(int[] arr, int n) {
    if(n==1) {
        return 0;
    }
    int res = arr[1]-arr[0];
    int min = arr[0];
    for(int i =1;i<n;i++) {
        res = Math.max(res, arr[i]-min);
        min = Math.min(min, arr[i]);
    }
    return res;
}
// Time Complexity - O(n)
```
