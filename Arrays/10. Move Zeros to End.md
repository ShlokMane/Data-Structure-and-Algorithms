Given an array move all the zero's present in the array towards the end. Maintain the sequence of other elements as it
is.

```
Example -
I/P - arr[] = [0, 10, 20, 0, 30, 40, 0, 50, 0]
O/P - arr[] = [10, 20, 30, 40, 50, 0, 0, 0, 0]
```

### Naive Solution

1. Create a temp array, and copy all the elements from given array to temp which are not 0. Then copy all the elements
   from temp back to the given array. Once copied all the remaining indexes in the given array would be 0. Time
   Complexity is O(n) and Space Complexity is O(n)
2. Traverse the given array whenever a 0 is encountered created another loop from that index and find a index where a
   non-zero element is present and swap this two elements. Time Complexity is O(n²) and Space Complexity is O(1).

### Efficient Solution

We maintain a variables while traversing the array. The variable signifies till where the array has non-zero elements.
If while traversing we find non-zero element we swap it with current element at the variable we are maintaing. 

```java
void moveZeroes(int[] arr, int n) {
   int count = 0;
   for (int i = 0; i < n; i++) {
      if (arr[i] != 0) {
         int temp = arr[count];
         arr[count] = arr[i];
         arr[i] = temp;
         count++;
      }
   }
}
// Time Complexity - O(n) and Space Complexity - O(1)
```
