Given an array return the index of the largest element in the array.

### Naive Approach

For every element i check all the other element if there is any element larger than it.

```java
int getLargest(int[] arr) {
    int n = arr.length;
    for (int i = 0; i < n; i++) {
        boolean flag = true;
        for (int j = 0; j < n; j++) {
            if (arr[j] > arr[i]) {
                flag = false;
                break;
            }
        }
        if (flag) {
            return i;
        }
    }
    // NOTE: this return statement will never be executed as there would a largest element present.
    return -1;
}

// Time Complexity - O(n²)
```

### Efficient Approach

While traversing the array we would keep a variable for the index of largest element till i, then compare if that is
smaller or greater than ith element.

```java
int getLargest(int[] arr) {
    int res = 0;
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] > arr[res]) {
            res = i;
        }
    }
    return res;
}
// Time Complexity - O(n)
```
