Given an array reverse the array. You have to reverse the same array don't create a new one.

We maintain two index `low` and `high`, one as 0 and other one as n-1 respectively. We swap elements at those indexes
and do `low++` and `high--`, we continue this until `low>high`.

```java
void reverse(int[] arr) {
    int low = 0, high = arr.length - 1;
    while (low < high) {
        int temp = arr[low];
        arr[low] = arr[high];
        arr[high] = temp;
        low++;
        high--;
    }
}
```
